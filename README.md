## 🎮 游戏介绍
BALENCIAGA智能挑战是一个交互式时尚游戏，包含两个回合：

### 第1轮：快速检索
模拟智能眼镜的快速信息检索功能，完成时尚产品的精确查询任务。

### 第2轮：需求转化
将用户的口语化描述转化为具体的产品识别，展示AI的理解能力。

## 🚀 快速开始
1. 点击上方链接进入游戏
2. 点击"START / 开始游戏"
3. 完成第1轮挑战
4. 点击"NEXT ROUND / 晋级"进入第2轮
5. 点击"RESTART / 重新开始"重玩
